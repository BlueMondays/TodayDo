# MBUtils

MBUtils은 Extension, Overriding 등 앱 개발시 필요한 util등을 모아둔 라이브러리 입니다.

이 라이브러리는 상시로 업데이트 되며, 개발시 필요한 요구사항이 있다면 모바일 앱 개발팀으로 알려주세요 : D
